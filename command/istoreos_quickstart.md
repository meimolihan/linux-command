istoreos_quickstart
===

iStoreOS 快速设置向导

## 补充说明

**quickstart** 是 iStoreOS 官方提供的快速设置向导工具，用于初始化配置网络模式、系统设置等。首次启动 iStoreOS 或需要重新配置网络模式时使用此命令。

### 语法

```shell
quickstart
```

### 功能说明

quickstart 向导提供以下配置选项：
- 网络模式选择（路由模式、旁路由模式、单臂路由等）
- LAN/WAN 接口配置
- 无线网络设置
- 系统基础设置
- iStore 应用商店初始化

### 常用实例

```shell
# 启动快速设置向导
quickstart

# 首次启动后配置网络
# 按提示选择网络模式：
# 1. 路由模式（默认）- 作为主路由使用
# 2. 旁路由模式 - 作为旁路由/网关使用
# 3. 单臂路由 - 需要配合主路由使用
# 4. 无线热点 - 作为 AP 使用

# 在向导中设置 LAN IP
# 输入新的 LAN IP 地址（如 192.168.1.1）

# 配置 WAN 口
# 选择 DHCP 自动获取或静态 IP
```

### 使用场景

```shell
# 场景1：首次启动 iStoreOS
# 1. 系统启动后执行 quickstart
# 2. 选择适合的网络模式
# 3. 配置管理 IP 地址
# 4. 完成初始化设置

# 场景2：切换网络模式
# 从路由模式切换到旁路由模式
quickstart
# 选择"旁路由模式"
# 设置 LAN IP 为与主路由同网段的 IP

# 场景3：重置网络配置
# 当网络配置混乱无法访问时
# 通过串口或控制台执行
quickstart
# 重新配置网络
```

### 配置流程

```bash
quickstart 向导流程：
┌─────────────────────────┐
│  1. 选择网络模式        │
│  2. 配置 LAN 接口       │
│  3. 配置 WAN 接口       │
│  4. 设置无线网络        │
│  5. 保存并重启网络      │
└─────────────────────────┘
```

### 网络模式说明

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 路由模式 | 作为主路由，WAN 接外网，LAN 接内网 | 家庭主路由、小型办公网络 |
| 旁路由模式 | LAN 口接入主路由，作为网关设备 | 已有主路由，需要特殊功能 |
| 单臂路由 | 单网口，通过 VLAN 划分 | 虚拟化环境、软路由 |
| 无线热点 | 仅作为无线 AP | 扩展无线覆盖 |

### 相关命令

```shell
# 查看当前网络配置
uci show network

# 查看网络接口状态
ifconfig

# 重启网络服务
/etc/init.d/network restart

# 查看 IP 地址
ip addr show
```

### 注意事项

1. 执行 quickstart 会中断当前网络连接
2. 配置完成后需要重启网络服务生效
3. 建议通过串口或直接连接方式进行配置
4. 配置过程中请确保电源稳定
